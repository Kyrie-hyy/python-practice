# Machine Learning Practice with Python

This repository contains my learning practice of Python and basic machine learning methods.
The project is intended to build a solid programming and machine learning foundation for further study in artificial intelligence.

## Contents
- Python basic syntax practice
- Numerical computing with NumPy
- Data processing with Pandas
- Classical machine learning classification using scikit-learn

## Machine Learning Experiment
A simple classification experiment is implemented on the Iris dataset using Logistic Regression.
This experiment includes data loading, model training, and evaluation.
